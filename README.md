# PatchLens

PatchLens is a native Qt-based Git patch viewer for Linux.

## Goals

- Open .patch and .diff files
- Side-by-side comparison
- Fast rendering
- Syntax highlighting
- Native Qt interface
- No Git repository required

## Planned Features

- File tree
- Hunk navigation
- Search
- Dark mode
- Word-level diff
- Patch statistics
